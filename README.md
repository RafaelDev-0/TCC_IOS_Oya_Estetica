# 🌿 Oyá Estética

Projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) no Instituto da Oportunidade Social (IOS), com o objetivo de criar uma plataforma moderna e intuitiva para uma clínica de estética, unindo presença digital, experiência do usuário e facilidade no agendamento de serviços.

## 📖 Sobre o projeto

O Oyá Estética foi desenvolvido para oferecer aos clientes uma experiência prática e agradável ao conhecer a clínica, seus profissionais e tratamentos estéticos disponíveis.

Além da apresentação institucional da empresa, a plataforma facilita o contato entre clientes e clínica por meio de informações organizadas, navegação intuitiva e recursos para agendamento de procedimentos.

## ✨ Funcionalidades

- 🏠 Página inicial com apresentação da clínica;
- 💆 Catálogo de tratamentos estéticos;
- 👩‍⚕️ Apresentação da equipe de profissionais;
- 📅 Área para agendamento de consultas;
- 📍 Informações de localização e contato;
- 📱 Interface responsiva para diferentes dispositivos;
- 🎨 Design moderno com foco na experiência do usuário (UI/UX).

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## 🎯 Objetivos

- Desenvolver uma solução web para uma clínica de estética;
- Aplicar conceitos de desenvolvimento front-end;
- Criar uma interface acessível e responsiva;
- Proporcionar uma navegação intuitiva para os usuários;
- Consolidar os conhecimentos adquiridos durante o curso.

## 📸 Telas do projeto

- Página Inicial
- Sobre a Clínica
- Tratamentos
- Equipe
- Agendamento
- Contato
