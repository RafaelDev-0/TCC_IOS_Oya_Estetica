# 🌿 Oyá Estética

Sistema desenvolvido como Trabalho de Conclusão de Curso (TCC), com o objetivo de modernizar e otimizar o gerenciamento de uma clínica de estética, oferecendo uma plataforma intuitiva para clientes e administradores.

## 📖 Sobre o projeto

O Oyá Estética foi desenvolvido para facilitar o agendamento de procedimentos estéticos, o gerenciamento de clientes e a administração dos serviços oferecidos pela clínica.

Além de proporcionar uma melhor experiência para os clientes, o sistema auxilia na organização da rotina administrativa, reduzindo processos manuais e aumentando a eficiência.

## ✨ Funcionalidades

- Cadastro e autenticação de usuários;
- Agendamento de procedimentos via whatsapp
- Interface responsiva.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap
